# 📊 Анализ продаж интернет-магазина 

## О проекте
Анализ данных продаж по 6 товарам, 5 регионам и 5 месяцам (январь–май 2025).  
Первый проект портфолио в рамках обучения BI / Data Analytics.

## 📁 Структура файла

| Лист | Описание |
|------|----------|
| Данные | Исходная таблица — 121 строка, 8 столбцов |
| Анализ | Выручка по категориям, месяцам и товарам |
| Дашборд | KPI, динамика продаж, ключевые выводы |

## 🛠️ Инструменты и формулы
- VLOOKUP — подтягивание цен из справочника
- SUMIF — выручка по категориям и товарам
- COUNTIF, IF, MONTH — вспомогательный анализ
- Сводные таблицы и срезы
- Условное форматирование
- Power Query — очистка данных

## 📈 Ключевые результаты

| Показатель | Значение |
|------------|----------|
| Общая выручка | 18 439 100 руб. |
| Количество заказов | 121 |
| Средний чек | 152 389 руб. |
| Топ категория | Электроника (85%) |
| Топ товар | Ноутбук (57% выручки) |

## 🔍 Выводы
- Электроника генерирует 85% выручки — основной драйвер продаж
- Ноутбук — самый прибыльный товар: более половины всей выручки
- Январь и апрель — лучшие месяцы, февраль и март — спад на 30–9%
- Мебель занимает только 15% — потенциал для роста
- Регионы: Москва и Санкт-Петербург лидируют по объёму заказов
